# Между строк

> Личный дневник с искусственным интеллектом.

«Между строк» помогает замечать закономерности в мыслях, эмоциях и жизни.

## Возможности

- ✍️ Дневник
- 🤖 Анализ ИИ
- 📊 Аналитика
- 🔍 «Что осталось между строк»
- 🔥 Серии записей
- ☁️ Синхронизация

## Стек

- React Native
- Expo
- TypeScript
- Supabase
- OpenAI
